# Arc 2: 噛み合わない

## メタ情報

- ID: arc-02
- Span: ep36–ep70（二年: 配役応用科）
- ステータス: scoped（中密度）
- テーマ: 他人と噛み合う。群像劇の本番
- 第一部テーマ（全3Arc通し）: 主役じゃないと自分が空になる恐怖を、主人公が初めて言語化するまで
- note: series-overview の密度ルールに従い、arc-02 は「四半期単位 + 主要イベント + 案件表」で構成

## Purpose

- この Arc で果たすこと: 「大きい話」「長い話」「人前の話」で共同配役を試み、構造的に失敗させる。固定化の誘惑を本格的に見せる。中点（学園＝ヒット作製造工場）を露見させる
- この Arc の終点: 久瀬が「自分が中心でないと空になる」恐怖を初めて言語化する。復権の噂が正式に流れる
- 次 Arc（arc-03 三年）へ渡すもの: 固定化リスクの具体例、工場の真実の輪郭、復権をめぐるクラスの分裂

## Main Turn

- 失うもの: 「小共同配役でなんとかなるかも」という一年目の自信
- 得るもの: 大きい話を回すには構造の問題があると理解する（まだ解決はしない）
- 反転点: 学園がヒット作製造工場であると知る（中点）

## 主人公の Span

- 開始状態: 一年目で「便利な駒切り」を一度止められた。でも大きい場になるとまた頭が走る
- 終了状態（arc-02）: 「自分が中心でないと空になる」を初めて口にする。計算はまだ走るが、それが自分の病だと自覚する

## ep36–42 案件表（二年目前半: 公開実習 → 長尺班案件 → 本格失敗）

> **位置づけ**: 一年目の仮答えを壊すブロック。「小共同配役の成功 → 公開実習の嫌な成功 → 長尺班案件の本格失敗」

### ep36「ノクス公開実習②」
- 舞台: ノクス・ランキング
- 案件: 公開実習後半。成功はするが"誰が何を吸って成功したか"を読者に見せる
- 痛点: 志摩（また埋める）、神代（前で拾う）。うまく回した二人だけが同じ席に座っている
- 前に出る: 神代、志摩、久瀬、百瀬
- 話の役割: 一年目の成功パターンが「大きい場では同じ顔に落ちる」と示す
- 終わり: 公開実習の評価は高い。高すぎる。平の顔だけが嫌そう

### ep37「ウケた、まずい」
- 舞台: 学園（評価会）
- 案件: 「成功したのに危ない」という学園の評価軸を見せる。二年目の不穏さを一段上げる
- 痛点: 成立＝高、見栄え＝高、協働＝低、固定化傾向＝警戒。久瀬（反論しきれない）、神代（笑って流そうとして顔が死ぬ）
- 前に出る: 平、久瀬、神代、志摩、七瀬、朽葉
- 話の役割: 学園の二枚舌（正しさと数字）を見せる
- 終わり: 次の案件発表。二年A組は長尺の班案件に上げられる

### ep38「長い話、向いてない」
- 舞台: 学園 → ネオン・アビス ブリーフィング
- 案件: 長尺班案件のルール説明。一回で終わらない、数週またぐ、観客の記憶が続く、敵役にも裏方にも継続性が必要
- 痛点: 久瀬（神代を前、志摩を裏、柊に泥──最悪の形がまた浮かぶ）
- 前に出る: 久瀬、柊、神代、志摩、比良坂、朽葉、百瀬、七瀬
- 話の役割: 最終山場の論点を小さめの形で再提示。「固定の顔」が欲しくなる構造
- 終わり: 転送開始。ネオン側は最初から"継続してバズる敵役"を求めている

### ep39「同じ顔を出せと言われた」
- 舞台: ネオン・アビス
- 案件: 長尺案件が自然に固定顔を欲しがることを見せる。一年目ネオンとの違い
- 痛点: 比良坂（固定した方が楽）、久瀬（否定しきれない）
- 前に出る: 神代、久瀬、志摩、比良坂
- 話の役割: 「話が長くなるほど、誰かを席へ縛りたくなる」構造の可視化
- 終わり: 志摩が明らかに疲れている。でも誰もまだその席を代われない

### ep40「便利だから、また」
- 舞台: ネオン・アビス
- 案件: 志摩固定化の危険を本格的に出す。柊の判断遅延がじわじわ効く
- 痛点: 志摩（毎回埋める）、百瀬（また？）、柊（間に合わない）、久瀬（気づいてるのに差し替えできない）
- 前に出る: 志摩、百瀬、柊、久瀬
- 話の役割: 「便利だからまた同じやつに頼る」の最悪さを積む
- 終わり: 神代が本番中に久瀬に言う──「お前、最初からそう組んでただろ」

### ep41「班案件、本格失敗」
- 舞台: ネオン・アビス
- 案件: 二年目前半の底。継続配信の山場で五人の悪癖が一斉に噛み合う
- 痛点: 久瀬（最悪の席を切る）、神代（前に出すぎ）、志摩（また埋める）、柊（判断遅延）、比良坂（切り捨て寄せ）、朽葉（正しい分析で追い込み）
- 前に出る: 久瀬、神代、志摩、柊、比良坂、朽葉
- 話の役割: 一年目ネオンよりはっきり「大きい話を共同で回す技術がまだない」と示す
- 終わり: 帰還時、神代と志摩の名前だけが別室呼び出しになる

### ep42「席に残される顔」
- 舞台: 学園（査定室・廊下）
- 案件: この12話の締め。神代/志摩が"固定候補"であると明確に示す
- 痛点: 神代（対外案件向け前景適性＝高）、志摩（補助・調整継続適性＝高）、久瀬（記録を見て吐きそうになる）
- 前に出る: 神代、志摩、久瀬、御門、雨宮
- 話の役割: "未来の先輩"を現実の進路として突きつける
- 終わり: 久瀬の内心──「同じ顔を前に置いて、同じやつに穴を埋めさせて、長い話を回す。それが一番楽で一番売れるなら。くそ。こっから先、本当にそこをやるのか」

### ep36–42 で達成すること

1. 一年目の成功が毒にもなると示す
2. 神代と志摩が「固定候補」として名前が上がる
3. 久瀬の主人公性が強まる（最悪の席を切れるが、それが最悪だとわかるねじれ）
4. 学園の不穏線が前に出る（まだ中点ではないが「うまくやるほど危ない」）

### ep36–42 ガードレール

- **やること**: 公開実習で「大きい場の歪み」体感 / 長尺班案件で本格失敗 / 神代と志摩の固定化リスク提示
- **やらないこと**: 中点（工場露見）はまだ / 復権を本格投入しない / 久瀬を反省させすぎない

## ep43–54 案件表（先輩・ローズ再訪・制度寄りの思想）

> **位置づけ**: 制度側の理屈に魅力と説得力を持たせるブロック。ここが弱いと中点以降の対立が浅くなる。

- 舞台: 学園（補講棟・ノクス公開実習の映像閲覧室）
- 案件: 御門を本格的に出す。神代にとっての"魅力的すぎる未来"

- 前に出る: 神代、久瀬、御門、平
- 話の役割: 固定の敵席が快楽と誇りとしても成立すると見せる
- 終わり: 御門が去り際に「お前、向いてるよ。残る側に」

### ep44「雨宮先輩、何でもやる」
- 舞台: 学園（裏方棟・実習準備室）
- 案件: 雨宮を本格的に出す。志摩にとっての"静かに終わる未来"
- 痛点: 志摩（見覚えがありすぎる動き）、百瀬（すごいのに寒い）
- 前に出る: 志摩、百瀬、雨宮

### ep45「残るのも仕事」
- 舞台: 学園（教室・面談室）
- 案件: 二つの未来を見たあとの反応を割る。クラス内に制度寄りの理屈が育つ
- 痛点: 久瀬（理屈だけなら制度側が一枚上）。柊・朽葉・七瀬が制度寄りに傾く

- 話の役割: 制度側がゼロから十まで間違っているわけではないと示す

### ep46「またあの庭へ」

- 案件: ローズ再訪の導入。今度は単発ではなく継続する恋愛劇
- 痛点: 固定の敗け席が自然に発生しやすい構造（百瀬=選ばれない側の親友、志摩=侍女/補佐、七瀬=祝福/格式、久瀬=側近/火消し、神代=前景）
- 前に出る: 久瀬、志摩、百瀬、七瀬、神代
- 話の役割: ローズを恋愛コメディではなく"綺麗な固定損"が成立しやすい場所にする
- 終わり: 現地に前からずっと身を引き続けている人物がいるとわかる

### ep47「ずっと選ばれない人」

- 前に出る: 百瀬、志摩、現地「選ばれない側」の人物

## ep67–70 案件表（復権の誘惑: 封筒〜ハーヴェスト再訪）

> **位置づけ**: 中点後の「誘惑を個人の問題へ落とすブロック」前半。ep67-70 は arc-02 最後の4話。復権の誘惑を "言葉" ではなく "実利" にし、クラスの空気が変わり始める。ハーヴェスト再訪で「復権しない未来」を一度見せる。

### このブロック（ep67-80）の4つの仕事

1. **復権の誘惑を"実利"にする**: 能力核再付与権という制度語ではなく、「戻れるかも」「元に近づけるかも」を現実化する
2. **固定席の魅力を具体化する**: 前景＝目立てる / 裏方＝迷わない / 継承＝品位と意味 / 観測＝間違えない、という"楽さ"
3. **クラス内の思想を割る**: 座るのも覚悟 / 一人で済むならマシ / 養分制度だろ / でも戻れるなら…
4. **久瀬に「最悪の設計」を一度やらせる**: 固定席を切る方が早いと理解し、実行してしまう → 主人公としての最低点

### ep67「封筒」
- 舞台: 学園（教室、個別ロッカー前）
- 案件: アウレリア帰還後の余波。復権の誘惑を個人通知で落とす
- 事件: 査定封筒が配られる。中身は人によって違う。神代（前景継続適性 高）、志摩（補助継続適性 高）、七瀬（継承・祝福継続適性 高）、柊（観測補助適性 高）、藍沢（能力核再査定候補）、弓月（長期安定案件適性 保留）。直接「復権できます」とは書かれていないが、継続席の方が明らかに能力核査定に近い
- 前に出る: 久瀬、神代、志摩、七瀬、藍沢、弓月
- 全体像への仕事: 誘惑を具体化し、クラスの空気変化を見せる
- 終わり: 久瀬が気づく。今日、誰も封筒の話をしない

### ep68「言わないやつが増えた」
- 舞台: 学園（凡日 / 寮・食堂・廊下）
- 案件: 誘惑がクラス内で"沈黙"になるのを見せる。凡日の微妙な空気
- 事件: 食堂でも寮でも誰も封筒のことを言わない。神代はいつも通り、志摩はもっと無言、柊は資料ばかり、百瀬は茶化せない空気にイラつく、冬城は「もらってない側」の居心地悪さ。久瀬はまた勝手に頭の中で席を切る
- 前に出る: 久瀬、百瀬、冬城、神代、志摩、柊
- 全体像への仕事: 誘惑は人を派手に壊すのではなく、まず沈黙にする、という手触り
- 終わり: 次の実習先は《ハーヴェスト・マーチ》再訪。低刺激・凡庸・平和な案件

### ep69「つまらない平和」
- 舞台: ハーヴェスト・マーチ（収穫祭準備）
- 案件: "凡人として生きる未来"を一度見せる。悪くないけどちょっと退屈
- 事件: 大事件なし。収穫祭の準備、荷運び、帳場、飾りつけ、見回り。ちゃんとやれば喜ばれる。誰も覚醒しない、誰も湧かない。藍沢は耐えられない、冬城は少し落ち着く、百瀬は「こういう終わり方もあるんだ」、久瀬は平和なくせに妙にざわつく
- 前に出る: 冬城、藍沢、久瀬、百瀬
- 全体像への仕事: 「復権しない未来」が地獄ではないと見せる。でも魅力的に見えない人もいる
- 終わり: 藍沢がぽつりと「これで終わるなら、戻りたいかも」

### ep70「戻りたいの、変か？」
- 舞台: ハーヴェスト・マーチ（収穫祭本番後）
- 案件: 復権の誘惑を"悪"ではなく"願望"として描く
- 事件: 藍沢の本音「地味にやるのはできる。でもこれで終わるのは嫌だ」。弓月も揺れる。冬城は平和で十分じゃないかと思う。久瀬は簡単に賛成も反対もできない
- 前に出る: 藍沢、弓月、冬城、久瀬
- 全体像への仕事: "戻りたい"を安直に否定しない。ここを認めておくから後の揺れが本物になる
- 終わり: 帰還後、次は《リフレイン・フロント》の長尺案件だと告げられる
