# Scene-Sequel Card

- packet_id: packet-001
- target_episode: ep01
- title: ちゃんと負けろ
- arc: 剥がされた凡人
- focal_character: 久瀬
- setting: 学園教室 / ノクス・ランキング闘技場
- story_job: 世界観導入。久瀬を主人公として立て、「この学園では主役を立てるために負けるのが仕事になる」を示す。
- core_wound: 主役食い。負けると自分の中身が空になる恐怖。

## Scene

- purpose: 世界観導入 + 久瀬主人公確立 + 「負けるのが仕事」という矛盾を読者に提示
- desire/goal: ノクス案件で「越えたくなる壁」として現地主人公・三科湊にきちんと敗ける
- obstacle/conflict: 久瀬は本能的に勝つ側・食う側へ寄る。三科は努力型主人公として真正面から食らいついてくるうえ、平の監査で「主役侵食」は即減点になる。久瀬は負ければいいのではなく、"相手の成長を成立させる形で負けろ"と言われ、それがひどく気持ち悪い。
- turn: 久瀬が一応敗けるが、敗け方がうますぎて観客の視線が三科ではなく久瀬へ流れる
- disaster: 案件は成立しつつも減点され、帰還後に「石像勤務」の処罰が下る
- reveal: 学園は再配役施設 / 痛ん者という俗称 / 教官は能力使用可・格上 / 実習では配役出力のみ使える / 久瀬は人を先に席で見てしまう（配役眼の端緒）
- emotional_turn: 久瀬は初めて、「負けるのが嫌」ではなく "意味のある負け方をさせられるのが嫌" だと気づく。屈辱が腹の中に残る

## Sequel

- reaction: 久瀬は「負けた」ことより、「意味のある敗北の部材にされた」ことに苛立つ。痛いというより腹が立ち、自分が気持ちよく勝てないとこんなに空っぽなのかと薄く気づく。
- dilemma: ここで反発して学園ルールそのものを拒むか、それとも屈辱を飲んで次の処罰に従うか。まだ久瀬には学園を壊す力も理由もない。
- decision: 久瀬は文句を飲み込み、石像勤務へ行く。少なくとも「この学園のやり方を知るまでは従う」しかないと決める。
