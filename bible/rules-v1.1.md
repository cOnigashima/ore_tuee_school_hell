# 文体・禁則 v1.1（差し替え候補）

> **status**: replacement-candidate
> **target**: `bible/rules.md`
> **intent**: 現行 `rules.md` の骨格を維持しつつ、本作固有のトーン定義・ボヤキ定義・シーン運用原則を上位概念として明文化した版

## 文体

- 視点: 三人称一元（scene ごとに focal character を固定）
- 時制: 過去形基調。回想は明確に過去完了で切る
- 一文の長さ: 短めを基調。哀愁のリズムに合わせて時々長い一文を混ぜる
- 地の文の密度: 薄めから中。描写の過剰は俺つえー的演出衝動に近づくので警戒する
- 情景描写の上限: 実習世界の描写は「配役の都合で見えている部分」だけ。世界観を語りすぎない
- 地の文の温度: 乾いていて少し皮肉。主人公の内面は禁断症状として身体反応で出す

## トーン定義（canon）

本作の文体は、**軽口を入口にし、生活感で人肌を出し、制度への皮肉で輪郭を立て、最後に小さな痛みを残す可変トーン文体**とする。

ただし本作は、コメディ・人肌・痛切・皮肉に加えて、**「嘲笑 / 優越 / 快楽」**を正面から扱う。元主人公たちが、他人を踏み台にし、見下し、主役を食うことに気持ちよさを感じていた事実を隠さない。その快楽を単なる悪として処理せず、一度は魅力として描いた上で、それが誰の固定損の上に成立していたかを示すことで、笑いを遅れて寒くする。

各シーンでは五要素のうち一つを主旋律とし、他の要素は後味として混ぜる。したがって、本作の感情は「平均化された中間トーン」ではなく、**主旋律が切り替わり続ける可変トーン**として運用する。

### 五つの主旋律

1. コメディ
2. 人肌
3. 痛切
4. 皮肉
5. 嘲笑 / 優越 / 快楽

### 強く使う遷移

- コメディ → 薄寒さ
- 嘲笑 / 優越 → 自己嫌悪
- 皮肉 → 生活感
- ほっこり → 未解決
- 痛切 → 軽口

## 視点制御

- POV の切り替え単位: scene 単位。同じ scene 内では切り替えない
- 地の文で書いてよい認知: focal character が正式に知っていること
- 書かない認知: focal character の知らない情報、他キャラの内面、語り手の全知的メタコメント
- 叙述の原則: **嘘は不可、省略は可**。情報操作は「順序」で行い、「嘘」で行わない

## 会話

### 温度（核）

- 「笑えるのに、少しだけ胸が痛い」を崩さない
- 慰め合うが、真正面から癒やしに行かない
- 和解で閉じない。少し本音が見えたところで、誰かが雑に切る
- 苦労話は「反省」ではなく禁断症状として書く

### ボヤキの定義（canon）

本作のボヤキは、他人を断罪するための悪口ではなく、**自分もその快楽に加担していたと知っている人間の、半歩遅れた自己嫌悪**として出す。

したがって、制度や他人を一方的に裁く言い方ではなく、

- 「それが気持ちよかったのもわかる」
- 「それが回るのもわかる」
- 「だから余計に嫌だ」

という二重感情を基本とする。

久瀬の独白では、ときに露悪的・優越的な言い回しを許容する。ただし、その快楽を**正義や正当性へ変換してはならない**。

### 皮肉の矛先（canon）

本作における皮肉は、個人の愚かさを安全圏から笑うためのものではない。人物を笑うことはあっても、最終的な矛先は、

- 席の偏り
- 役の固定
- 綺麗な物語の裏で発生する損失
- 優越の快楽を安定運用できてしまう制度

へ向ける。

人物への嘲笑で終わらず、**「なぜそれが回ってしまうのか」**まで描いて、はじめて皮肉として成立する。

### ほっこりの定義（canon）

本作における「ほっこり」は、関係の温度や小さな救いを示すためのものであり、**問題の解決や和解の確定を意味しない**。

あたたかい場面のあとには、必ず

- 未解決のズレ
- 疲れ
- 制度の影
- 次に来る不安

のいずれかを少しだけ残す。

### 「くそ」切りの話法（canon）

- 本音に触れそうになったとき、登場人物は「くそ」で話を切る
- 「くそ」は感情の噴出口であり、会話を終わらせる機能を持つ
- 乱発しない。1 scene に 1 回を上限の目安にする
- 「くそ」以外の切り方: 「もう寝る」「帰る」「うるせえ」「それは正常だ」など

### 対話の目的

- すべての台詞は次のいずれかを担う: ①情報伝達 ②関係性の変化 ③欲望の衝突
- どれも担わない台詞は削る

### 説明セリフの上限

- 1 scene に 1 回まで
- 制度や世界観の説明は、教官の査定語か処罰の現象で回収する

## シーン運用原則（canon）

各シーンは、コメディ・人肌・痛切・皮肉・嘲笑 / 優越 / 快楽のうち一つを**主旋律**として選ぶ。全要素を均等に混ぜることは避け、主旋律を前に出したうえで、終わり際に別の要素を一滴だけ残す。

### 運用手順

scene 着手前に以下を決める。

1. この scene の主旋律は何か
2. 誰の古傷を刺すのか
3. 終わり際にどの後味を残すのか

## 独白の温度（canon）

- 主人公の独白は「嫌な理屈」のトーン
- ただし嫌な理屈だけでなく、**昔はそれが気持ちよかった**という快楽の記憶も混ぜてよい
- 合理化語彙（「つまり」「要するに」等）が続いたら削る
- 観念だけで閉じず、身体反応と並走させる

## 口語化ガイド（canon）

- 主語を抜く
- 一文を短くする
- 途中で切る
- 同じことを二回言わない
- 感情は滲ませる
- 本音ほど短くする

## 会話温度のモード表（canon）

| 場面 | 温度 |
|---|---|
| 実習前 | だるい、雑、学生っぽい |
| 実習中 | 短い、仕事口調 |
| 帰還後 | 軽口多め、でも痛い |
| 深夜の寮 | 本音が少し出る |
| 教官面談 | 冷たい、逃げ場がない |
| 祭り回や変な案件 | かなり笑える |

## 絶対 NG

- 役割語テンプレの貼り付け
- 元主人公たちをただの愚か者として嘲笑うだけの書き方
- 快楽や優越を全部なかったことにすること
- 和解・反省・改心で会話を閉じること
- 俺つえー批判を説教として書くこと
- 地の文での全知的なメタコメント

## 避ける

- 過剰な情景描写
- 説明セリフの連発
- 恋愛を「救い」として扱う描写
- 「くそ」の乱発
- 全シーンを同じ乾いた温度で通すこと

## 最短要約

本作は、**笑って、見下して、気持ちよくなって、そのあと少し寒くなる**文体を目指す。ただし、その笑いも優越も安全圏からのものではなく、**自分もそこにいたと知っている者の笑い**として運用する。
