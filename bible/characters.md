## クラス構成（症状別）

学園では「不良 / 優等生」ではなく、**主役癖の再発しやすさと症状**でクラスを分ける。全員が完全矯正されていたら話が死ぬ、という前提。

### A 組：実務適応組

- 状態: ちゃんと負けられる。引ける。主役を食わない
- ドラマ機能: 「譲れた者の哀愁」を担う。模範ではなく、諦めの先にある静けさ
- 再発リスク: 低（だが、まれに"試してみたく"なる）

### B 組：再発監視組

- 状態: まだ見せ場を奪いたがる。決め台詞を足したがる
- ドラマ機能: 物語のコメディと葛藤の主戦場。主人公候補はここから出る可能性が高い
- 再発リスク: 中〜高

### C 組：役割喪失組

- 状態: 主役じゃないと何もできなくなる。無力感が強い
- ドラマ機能: 「奪われた中心感」をそのまま抱える。静かな自傷の温度を担う
- 再発リスク: 低（だが崩壊リスクあり）

### 特別矯正組：世界観汚染常習者

- 状態: 他作品のノリや技名を勝手に持ち込む。ハーレム形成、無断覚醒、脚本破壊
- ドラマ機能: 最も露悪的なコメディを担う。読者の"俺つえー記憶"を直撃する役
- 再発リスク: 高。常時監視対象

## 教官類型（演出衝動の矯正官）

教官は「反抗心」ではなく「自己中心的な演出衝動」を矯正する。道徳家ではない。

### 負け方を教える教官

- 担当: 配役として美しく負ける技術。祝福して退場する呼吸
- 口癖サンプル: 「一騎打ちで負ける方がまだ楽だよ。祝福して退場しろって言われた」（これは生徒側の台詞例だが、教官の教材に出てくる）

### 目立たない演技を教える教官

- 担当: 背景役として視線を集めない訓練

### 恋愛ルートへの介入を防ぐ教官

- 担当: 配役中の関係形成を管理する。恋愛が配役を乱すため

### 借り物の決め台詞を検出する教官

- 担当: 世界観汚染の一次検出。他作品ノリの流入を防ぐ

### "主人公を潰さず追い込む"圧の作り方を教える教官

- 担当: ライバル役・四天王中堅役のための圧の設計

### 教官の共通口癖

- 「矯正するのは反抗心じゃない。自己中心的な演出衝動だ」
- 「お前は反省していない」「してます」「違う。お前は"反省している自分"を演出している」
- 「矛盾に気づいているから偉いと思うな」「自覚してなお同じことをやるなら、それは症状じゃない。癖だ」

## char-001 [久瀬（くせ）: 主人公・ルーター型]

- name: **久瀬**（くせ）。下の名前は未決定
- role: protagonist
- former_engine: **ルーター型**（鑑定で相手の弱点と最短手を特定 → 自分の技を最適化 → コピー → 演出で仕上げる複合エンジン）
- former_cheat_name: **《模倣最適化》**（見た相手の技・立ち回り・口上・人を惹きつける間・主人公っぽさそのものを、自分が一番映える形に再構成して使う。ただのコピーではなく「自分が映える形」にする点がえげつない）
- former_title: 未決定（かつて持っていた二つ名は禁句）
- residual_habit: **配役眼**（スキルではなく後遺症としての悪癖。人を見ると先に"役名"が浮かぶ──壁役、宿敵、負けヒロイン、便利な補佐、炎上要員、犠牲にしやすい役。便利さでもあり最低さでもある）
- reader_function: **入口**（読者は久瀬の配役眼を通して世界構造を理解する。久瀬=入口、志摩=良心、神代=快楽、柊=不安）
- why_protagonist: テーマの罪を一番背負っている（他人のかっこよさを借りて主人公化してきた＝学園に送られるべきだった男）。ルーターとして読者のフィルター性能が高い。有能だった中身が借り物だらけ＝「俺には元から何もなかったんじゃないか」の恐怖
- story_function: 「譲れない自分」を演じ続ける苦しさを読者に渡す
- public_face: 矯正を受け入れた優等生のふり
- private_truth: 主役でいた頃の癖と誇りを手放せない。剥奪された UI の輪郭を、今も視界の端で探してしまう
- character_direction: 正しい人にしない（良心は志摩の仕事）。口数は多すぎず内心は俗っぽく（口悪い観察者）。早い段階で"凡人だ"を見せる（喧嘩強くない、カリスマない、体力ない、特別優しくもない）
- external_goal: 矯正を表向きクリアしていく
- internal_lack: 自分が中心でない自分を受け入れられない
- false_belief: 「自分が中心でなければ、自分は何者でもない」
- fear: 名前のある役から削除されて、完全に部材になること
- desire: 「見せ場」をもう一度掴みたい
- need: 他人の物語を成立させたという、派手ではない誇り
- contradiction: 譲りたい／でも譲ると空っぽになる
- secret: ルーター型は「自分で何かを生み出した」ことが一度もなく、全部借り物の合成だったという自覚
- pressure_response: 本音に触れそうになると「くそ」と話を切る
- moral_line: 配役先の現地主人公を潰すことは、どれだけ誘惑があってもやらない
- monologue_tone: 「嫌な理屈」。言い訳と本音が混ざる。合理化語彙に引きずられやすいので drafter は警戒
- signature_monologue: 「負けるのが嫌なんじゃない。ちゃんと意味のある負け方をさせられるのが嫌なんだ」
