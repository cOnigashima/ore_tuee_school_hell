## 舞台

- 時代: 少しだけ自然主義を捨てた配役世界。人間が石像・火花・女神の声まで担当しうる
- 場所: **幕間域**（各物語世界のあいだに存在する"裏方空間"）に学園がある。どの世界にも属さず、実習世界へゲートで接続する
- 社会の雰囲気: 労働と矯正が建前、裏では元最強たちの禁断症状と哀愁が漂う

## 基本構造（配役世界）

- 学園は本舞台ではなく、訓練と矯正の拠点。本当の舞台は派遣先の"物語空間"
- 生徒は各種の実習世界へ派遣され、配役として働く
- 配役の粒度は 4 段階: 人物役 / 背景役 / 装置役 / 演出役
- 配役が人間を部材化する理屈は "少しだけ自然主義を捨てる" ことで成立させる。ただの魔法学園ではない

## 呼称（canon）

- 正式呼称: **異端者**（いたんしゃ）。矯正対象の元・無双主人公を制度上こう呼ぶ
- 俗称: **痛ん者**（いたんしゃ）。生徒どうしの隠語。自虐と親しみの両義
- 同音異字で書き分けるのが canon。地の文では文脈で使い分ける

## 入学者の定義（canon）

入学者は、破綻・暴走・過剰適応・削除対象になった元主役/元準主役/元固定席候補。単に悪人だけが来るわけではない。「うますぎて危険」「使い潰しの構造に適応しすぎた」者も対象。これで御門・雨宮のような先輩も自然に存在する。

## 入学（収容式）の剥離（canon）

- 入学時に**能力核**を封印する。封印対象:
  - チートスキル・固有魔法
  - 加護/祝福
  - 肩書（勇者／魔王／聖女 等）と役職固定補正
  - 死に戻り/セーブ＆ロード
  - 鑑定UI・ステータス画面・称号欄
  - 有能相棒リンク（聖剣・相棒獣・従者・精霊）
  - 有能部下指揮補正
  - 並列思考/高速思考
  - 無限アイテム/召喚枠
- 前提: **チートや能力がなければ、ただの凡人**
- 例外: 教官のみ「チート由来の矯正能力」を限定保持（症状検出・配役強制の用途に限る）
- 収容式は入学式と同一。形式は葬式に近い
- 能力核は学園側が保管。返却条件は公開されていない

## 出力貸与システム（canon）

- 学生は自前のチートを使えない。実習では案件ごとに**配役出力**だけ一時貸与される
- 昔の強さをそのまま持ち込むことはできない
- 役として必要な最低限の性能（身体強度・視力・声量）だけ貸与される
- 勤務終了とともに回収される
- 貸与は「チート復活」ではなく「部材としての最低動作保証」
- 貸与量が意図的に不足している配役もある（使い捨て勤務など）
- だから問われるのは強さより運用。「働く」感じはここから出る

## 校務権限（教職員の能力運用 canon）

- 学生: 入学時に全剥奪
- 教職員: **今も能力を使える。格上。** 校務権限として保持
  - 昔のチートそのままではなく、管理・査定・差し戻し・拘束の嫌な強さ
  - 気持ちいい戦闘無双ではない。逃げ場のない管理能力
  - 学生との権力差と恐怖を出す道具
- 教官権限の原則:
  - 学園内で自由に行使できる
  - 実習中は監督・差し戻し・緊急介入に限る
  - 能力核の返還・完全封印は学園長級のみ
  - **教官単独では、学生を永久固定席へ確定できない** → 査定会議と記録承認が必要
- 学園長級: 剥離・封印・返還・門管理・最終案件割当を持つ唯一の権限。学生の能力を奪えるのはこのレベルだけ。教官は強いが、制度の根幹そのものではない
- 本文で先に能力説明をしない。嫌な精度で見せて、あとで噂にする

## 能力核再付与権（復権）（canon）

- 正式名称: **能力核再付与権** ── 書類・校内掲示でのみ使う堅い制度語
- 教師と生徒の会話: **復権**
- 砕けた言い方: **元に戻れる権利** / **返してもらえるやつ**
- 本文で毎回正式名称を使わない。レジスターで書き分ける
- 最終山場での誘惑として機能する。これを全員で捨てるから最後が効く

### 復権の三段階（canon）

| 段階 | 名称 | 内容 |
|---|---|---|
| 第一段階 | **校務補助権** | 限定された場で、能力核の一部機能を補助的に使える |
| 第二段階 | **準復権** | 卒業後または継続案件内で、能力の一部を再解放される |
| 第三段階 | **全復権** | ごく稀に、元の階層に近い形で能力核が大きく戻る |

- 学園はこの三段階をまとめて**「復権」と呼んで釣る**。これが制度のいやらしさ
- 生徒から見えるのは「戻れるかもしれない」。実際は、固定席に長く座った者ほど近づく

## ハードルール（できないこと・代償優先）

- **できないこと**:
  - 生徒は配役中に自分の意思で見せ場を作れない（役割破壊として検出される）
  - 他の作品世界のノリや技名を勝手に持ち込めない（世界観汚染として罰せられる）
  - 恋愛は配役を乱すので、勤務中の接触は制限される
- **代償**:
  - 見せ場を奪うと背景勤務、演出に酔うと演出素材勤務、神や運命を都合よく使うと神話勤務に落ちる
  - 一撃必殺で物語を短絡させた者は使い捨て勤務（弾頭・囮・格上げ要員）に回される
- **例外**（最大 1 つまで）: 未決定。2 つ目を追加するときは `story/design-debt.yaml` に記録する
- **禁忌**: 配役中に「俺が主人公だ」という演出衝動を通すこと

## 3 層警告システム（canon）

配役破壊の兆候に対して、処罰の前段として 3 層で警告が飛ぶ。

1. **個人通知**: 生徒の視界に矯正官の声や文字が割り込む（内なる声、視界の端のテロップ）
2. **世界反応**: 実習世界自身が配役破壊を検出し、物理現象（膝の石化、指先の火花化、声のエコー化）で警告する
3. **帰還後評価**: 学園に戻ってからの面談・減点・勤務シフト変更

1 話に 3 層すべてを詰める必要はないが、長期的に 3 層を揃えて見せる。

## 実習中の痛み・死・残響（canon）

### 痛み
痛みは**完全に本物**。実習中の疲労・恐怖・苦痛はそのまま感じる。

### 怪我
- 軽傷: 帰還時にある程度リセット
- 重傷: 帰還後も数日〜数週間、鈍痛や可動制限が残る
- 精神的負荷: 長く残る

### 死
学生は実習中に死ぬと、**幕間域へ強制帰還**する。永久死ではない。ただし代償がある:
- 激しい**断線痛**
- 評価大幅減点
- 数日単位の行動制限
- 繰り返すと**残響**が残る

### 残響（canon）
残響は、何度も同じ痛みや死を引き受けた結果残る**後遺症**。
- 幻痛
- 悪夢
- 役に引きずられる（配役を降りても口調や感情が戻らない）
- 特定の席に"馴染みすぎる"

これで、主損席や固定席の怖さが制度だけでなく**身体にも**出せる。

### 現地キャラの死
現地世界の人間は基本的に**本当に死ぬ**。実習世界はただの幻ではない。この作品の倫理が立つ根拠。

## 実習世界の時間と記憶（canon）

- 実習世界の時間は**継続して流れる**。学園は"節点"に介入できるが、基本的には巻き戻せない
- 再訪したら前回の結果が残っている
- 現地レギュラーは覚えている
- だから案件は積み重なる
- 例外: 死に戻り系などその世界自体がループ構造を持つ時だけ

## 成仏点 / 供養点（二重評価 canon）

- **成仏点**: 表側の冷酷な評価指標。配役として美しく消費されたかを採点する（配役管理部門が管理）
- **供養点**: 裏側の救済指標。学園スタッフが非公式に積む「この異端者は、まだ人間として扱われている」という記録
- 二重帳簿は公然の秘密。生徒は成仏点しか見えないが、教官は両方を見ている

## 処罰体系（役割破壊に対する処罰）

**設計基準（canon）**: その処罰は、その人物が昔に奪っていたものの側に落ちているか。

### 背景勤務

- 神殿の石像 / 玉座の間の甲冑 / 学園の銅像 / 祝祭の群衆 / 王都の通行人 / 主人公の勝利を見守る無名兵

### 演出勤務

- 必殺技の火花 / 斬撃の軌跡 / 爆煙 / 稲妻 / 覚醒時の光 / 勝利演出の風

### 神話勤務

- 女神の声 / 神託 / 聖剣の台座 / 封印の石碑 / 祝福の光 / 教会壁画の聖人

### 使い捨て勤務

- 決め技の弾頭 / 囮 / 開幕一発で吹き飛ぶ中堅 / 主人公の格上げ要員 / 「こいつ強え！」と言うだけの先輩

### 因果の例

- 神の加護で無双してたやつ → 女神役
- 必殺技連発で格を潰してたやつ → 火花・爆煙
- 世界の中心ぶってたやつ → 背景石像
- 一撃で全部終わらせてたやつ → 弾頭

## 三学年制（canon）

年齢ではなく**在籍年次**で区切る。年齢はバラバラ。

### 一年: 剥離基礎科

- テーマ: 「凡人として立つ」
- 内容: 能力のない日常に慣れる。個人案件をこなす。負け方・退場・モブ会話を学ぶ
- 案件比率: 個人 60 / ペア 30 / 班 10

### 二年: 配役応用科

- テーマ: 「他人と噛み合う」
- 内容: ペア案件。恋愛導線。パーティもの・追放もの・配信ものの班案件
- 案件比率: 個人 25 / ペア 40 / 班 25 / 長編 10

### 三年: 卒業実習科

- テーマ: 「固定生贄なしで長編を成立させる」
- 内容: 長編案件。公開案件。学年横断案件。卒業実習
- 案件比率: 個人 10 / ペア 20 / 班 40 / 公開長編 30

### 留年（canon）

- 勉強不足ではなく、**役に馴染みすぎて残される**
- 留年理由は主に4つ:
  1. **主役食いが抜けない**
  2. **固定席適性が高すぎて離されない**
  3. **協働不能**
  4. **能力なしの日常適応に失敗**
- 「うまいほど危ない」が成立する
- 敵役がうますぎて卒業させてもらえない / 支え役が便利すぎて残留 / 調整役として制度に取り込まれる
- だから先輩が怖い。「うまくやりすぎた完成形」であり「壊れた姿」でもある

### 在籍区分（canon）

| 区分 | 意味 |
|---|---|
| **在籍** | 通常学生 |
| **進級** | 次学年へ |
| **留年** | 単位不足 or 固着指数過多 |
| **仮継続席候補** | 固定席候補として査定中 |
| **継続在籍** | 卒業延期で固定席運用に入る状態 |
| **卒業解放（成仏）** | 学園の配役管理から解放 |

### 進級・卒業

- 進級判定は年度末（3月）
- 卒業条件: 最終実習（三年次の長編案件）を「固定生贄なしで」成立させる
- 卒業式は **成仏式**

### 成仏の意味（canon）

成仏 = 学園の配役管理からの**解放**。
- 能力核は返還されないまま、物語の固定席がない生活圏へ移る
- 元の"主役保護"つき世界には戻らない
- **記憶は保持**
- **能力は戻らない**
- ただの一人として生きる権利を得る
- 俗称「成仏」、正式名称は**卒業解放**
- 希望すれば、**同じ期の仲間と同じ生活圏へ行ける**。これでラストの「全員で成仏」が成立する

## 案件の4段階（canon）

1. **個人案件**: 一人だけが配役される。痛点を刺すのに最適
2. **ペア案件**: 二人組。相性の悪い組み合わせで関係が深まる
3. **班案件**: 3〜5人。チームワークと事故が出る
4. **クラス案件/長編案件**: クラス全体で一つの物語世界を支える。山場用。一学期に一回あるかないか

## 週の流れ（canon）

| 曜日 | 内容 | 物語上の機能 |
|---|---|---|
| 月 | 案件通達、役割説明、授業 | Goal を置く日 |
| 火 | 一次実習（個人/ペア/班） | 衝突が始まる日 |
| 水 | 帰還、査定、授業、処罰 | 結果が出る日 |
| 木 | 再実習、補講、役替え、班演習 | Reaction / Dilemma |
| 金 | 評価確定、次の組み合わせ発表 | Decision を置く日 |
| 土 | 自主訓練、観覧実習、学内イベント | 本筋以外の関係を作る日 |
| 日 | **凡日** | 休み。何も起きない時間に耐える訓練日 |

### 凡日（canon）

- 外出、洗濯、買い物、食堂、寮、だらつき、デート未満
- 何もしない時間に耐えられない元主人公たちにとって、凡日がいちばん苦しい
- 昔はイベントが勝手に起き、ヒロインが寄ってきて、戦闘や覚醒が発生して生活が飛んでいた
- 「何も起きない一日をちゃんと過ごす訓練日」

## 授業（canon）

普通教科ではなく、**配役実務と矯正**の授業。

- 負け方実技
- 退場作法
- モブ会話演習
- 悪役学
- 恋愛導線管理
- 尺管理
- 口上矯正
- 世界観読解
- 背景・群衆・エフェクト実習
- 生活適応（買い物、洗濯、金銭感覚）

「授業でしかできない、しょうもない訓練」が作品の味になる:
- 5分間、主人公を見ても意味深な顔をするな
- モブとして自然な雑談を続けろ
- 驚き役として過不足ないリアクションを取れ
- 三回連続できれいに負けろ

## 評価4本柱（canon 更新: 成仏点/供養点を統合）

### 表4柱（生徒 + 教官に見える）

| 柱 | 内容 |
|---|---|
| 配役点 | 役割を守れたか |
| 成立点 | 物語を成立させたか |
| 協働点 | 他人を使い潰さずに連携できたか |
| 生活点 | 能力なしの生活を破綻させていないか |

### 裏2指標（教師側だけが見える）

| 指標 | 内容 |
|---|---|
| **固着指数** | 特定の席に馴染みすぎていないか |
| **他者固定指数** | 他人を固定席へ押し込みやすいか |

- **久瀬が危ないのは他者固定指数**。表向き優秀でも将来的に危険扱い
- 成仏点: 表4柱の総合スコア（表の顔）
- 供養点: 裏指標の逆数的な存在。「まだ人間として扱われている」記録（裏の顔）

### 供養祭（文化祭 canon）

- 単なる学校イベントではない
- **"固定席なしで短い物語を成立させられるか"を試す公開演習**
- 性格:
  - 校内イベント
  - 他学年・教官・一部外部観覧あり
  - クラス単位の連作・短編・実演が主流
  - 配役の供養、没役の再利用の意味もある
- 最終山場の縮小版として機能する

## 学校の規模（canon）

- 本編の舞台は**配役科 A組**。各学年1クラス、少人数制
- ただし以下は存在する:
  - 補講棟
  - 継続席運用棟
  - 他科・他班
- 世界の広がりはあるが、**本筋は1学年1クラスの少人数で追う**。これで100話でも散りすぎない

## 経済・寮・生活（canon）

### 生活費
- 食事・寮・医療は学園持ち
- 小遣いは**生活点 + 案件手当**で支給
- 減点や処罰で小遣いは減る（これで冬城が機能する）

### 外出
- 凡日に一定条件で許可
- 処罰中は制限
- 他世界の通貨やアイテムは持ち帰れない

### 寮
- 一・二年は原則**二人部屋**
- 三年は準個室も可
- 具体的な部屋割りは本文制作時に決定

## 日常（ベースキャンプとしての学園）

- 衣食住: 寮生活
- 仕事: 授業＋矯正訓練＋実習派遣
- 移動: 実習世界への派遣は学園が管理する転送システム（詳細未決定）
- 通信: 派遣先との連絡手段は未決定

## 勢力 / 制度

- 学園職員（演出衝動の矯正官）: 「反省ではなく症状」を扱うスタンス
- 配役管理部門（仮）: 派遣先と配役を割り当てる
- 生徒（元・最強主人公たち）: クラス制。詳細は `bible/characters.md`
- 外部の"物語空間": 無数にあり、実習先として消費される

## 実習世界（7 つに固定 canon）

同じ世界でも役を変えれば別の話になる。世界の再利用は武器。最終案件《エバーアフター》は別枠（7世界の合成版）。

### 運用ルール（canon）

1. **再訪ルール**: 各世界は2〜4回再訪する。毎回仕事を変える
   - 初回: テンプレ紹介 + 痛点を刺す
   - 2回目: 役割反転 + 関係悪化
   - 3回目: 思想対立 + 制度の本音
   - 4回目: 最終局面の反響
2. **外側レギュラー上限**: 1世界につき最大3人（現地主人公 / 固定生贄になりやすい役 / 圧を体現する第三者）。これ以上増やすと学園側の群像がぼける
3. **実習:学園 = 3:1**: 3話実習したら1話は学園内に戻す（授業、寮、凡日、処罰、買い物、供養祭準備）。これを入れないと能力を失ったあとの"普通の痛さ"が消える
4. **学年別意味変化**: 同じ世界でも学年ごとに意味を変える
   - 一年目: 痛い
   - 二年目: 噛み合わない
   - 三年目: 選ばされる
5. **始点・中点・終点への接続**: 全てのアークは始点（凡人の屈辱）→ 中点（学園=工場）→ 終点（復権放棄+共同配役）に接続する

## 永久ヒット案件（第一部の山場 canon）

- 配役管理部門が運営する裏制度。特定のヒット作を永続させるため、固定の生贄役を永久に配置する
- 例: ある世界の「負けヒロイン」を、同一人物に永久に担当させ続ける
- 表向きは「適性配役」、実態は「使い捨ての固定化」
- 第一部のクライマックスはこの制度の発見 → 告発 → 解体
- 最終犠牲: クラス全員が「元チートに戻れる可能性」を手放す選択を下す

## 確定設定

- タイトル: 『元オレつえー学園 ──かつて最強だった俺たちは、かませ犬として働きます』
- 世界は配役世界であり、学園はベースキャンプ
- 学園は**幕間域**（各物語世界のあいだの裏方空間）に存在する
- 処罰の設計基準は「昔に奪っていたものの側に落ちるか」
- 第二保健室は破棄（seed 内で決定）
- 入学時にチート・肩書・相棒・UI をすべて剥奪する（学園長級の権限）
- 入学者は破綻・暴走・過剰適応・削除対象の元主役/元準主役/元固定席候補
- 教官は今も能力使用可。格上。嫌な管理能力。ただし教官単独では永久固定席確定不可
- 学園長級のみが剥離・返還・門管理・最終案件割当を扱える
- 三学年制（在籍年次）。一年=剥離基礎、二年=配役応用、三年=卒業実習
- 在籍区分6種（在籍/進級/留年/仮継続席候補/継続在籍/卒業解放）
- 留年は「役に馴染みすぎて残される」。4理由（主役食い/固定席適性/協働不能/日常適応失敗）
- 成仏 = 卒業解放。能力不返還、記憶保持、同期と同生活圏可
- 復権は三段階（校務補助権/準復権/全復権）。学園はまとめて「復権」と呼んで釣る
- 案件は4段階（個人/ペア/班/クラス長編）
- 週の流れ: 月通達→火実習→水帰還査定→木再実習→金評価→土イベント→日凡日
- 授業は配役実務＋矯正（負け方実技、退場作法、モブ会話演習 等）
- 評価: 表4柱（配役/成立/協働/生活）+ 裏2指標（固着指数/他者固定指数）
- 評価は二重帳簿（成仏点／供養点）
- 行事: 4月剥離式/6月役替試験/7月中間実習/10月供養祭/3月成仏式
- 供養祭 = 固定席なしで短い物語を成立させる公開演習
- 実習世界は7つを再利用（異能序列学園/追放ギルド/王立学院/ダンジョン配信/神殿国家/スローライフ村/戦線）
- 実習世界の時間は継続。再訪したら前回の結果が残る。現地キャラは本当に死ぬ
- 痛みは本物。死は幕間域へ強制帰還（断線痛+減点+行動制限）。残響が蓄積する
- 呼称は「異端者」（正式）／「痛ん者」（俗称）で書き分ける
- 警告は 3 層（個人通知／世界反応／帰還後評価）
- 学校規模: 配役科A組、各学年1クラス、少人数制
- 経済: 食事/寮/医療は学園持ち。小遣いは生活点+案件手当。寮は一二年二人部屋、三年準個室可
- 第一部の最終山場は「永久ヒット案件」の解体
