# 文体・禁則

> **source seed**: `story/seeds/2026-04-09-moto-oretuee-gakuen.md`
> **rule**: 文体指針と禁則の正本。drafter はここを読んでから draft を書く。

## 文体

- 視点: 三人称一元（scene ごとに focal character を固定）。多元は当面使わない
- 時制: 過去形基調。回想は明確に過去完了で切る
- トーン: 笑えるのに、少しだけ胸が痛い。露悪にはしない
- 一文の長さ: 短めを基調。哀愁のリズムに合わせて時々長い一文を混ぜる
- 地の文の密度: 薄めから中。描写の過剰は俺つえー的演出衝動に近づくので警戒する
- 情景描写の上限: 実習世界の描写は「配役の都合で見えている部分」だけ。世界観を語らない
- 地の文の温度: 乾いていて少し皮肉。主人公の内面は"禁断症状"として身体反応で出す

## 視点制御

- POV の切り替え単位: scene 単位。同じ scene 内では切り替えない
- 地の文で書いてよい認知: focal character が正式に知っていること（drafter-preflight の知識状態台帳と同期）
- 書かない認知: focal character の知らない情報、他キャラの内面、語り手の全知的メタコメント
- 叙述の原則: **嘘は不可、省略は可**。情報操作は「順序」で行い、「嘘」で行わない

## 会話

### 温度（核）

- 「笑えるのに、少しだけ胸が痛い」を崩さない
- 慰め合うが、**真正面から癒やしに行かない**
- 和解で閉じない。少し本音が見えたところで、誰かが雑に切る
- 苦労話は「反省」ではなく**禁断症状**として書く

### 「くそ」切りの話法（canon）

- 本音に触れそうになったとき、登場人物は「くそ」で話を切る
- 「くそ」は実質、感情の噴出口であり、会話を終わらせる機能を持つ
- 乱発しない。ここぞで打つと効く。1 scene に 1 回を上限の目安にする
- 「くそ」以外の切り方: 「もう寝る」「帰る」「それは正常だ」「うるせえ」など、癒やしに踏み込まない雑な切断

### 対話の目的

- すべての台詞は次のいずれかを担う: ①情報伝達 ②関係性の変化 ③欲望の衝突
- どれも担わない台詞は scene card に戻して削る

### 口調の方針

- 役割語テンプレの貼り付けを禁じる
- 台詞の個別化は「価値観／優先順位／禁句」から設計する（`bible/characters.md` 参照）
- 語尾の差別化より、話を"どこで切るか"で個性を出す

### 説明セリフの上限

- 1 scene に 1 回まで。それ以上必要なら演出勤務の矯正官に喋らせる（世界観説明は教官の口癖で回収する設計）

### ギャグ / シリアスの比率

- 基本は 7:3 でギャグ寄り、ただし "ここぞ" の哀愁は押し引きで立てる
- シリアスは 1 エピソードに 1 箇所の"胸が痛いポイント"で集中する

## 固有名詞と表記

- 固有名詞の正式表記: 「元オレつえー学園」（"俺"は片仮名"オレ"で固定）
- 配役関連語: 「勤務」「配属」「派遣」「背景役」「演出素材」「神話勤務」「使い捨て勤務」
- 呼称: 正式は「異端者」、俗称は「痛ん者」。同音異字で書き分ける（canon）
- 制度語: 「成仏点」（表）／「供養点」（裏）／「収容式」（入学式）／「出力貸与」／「永久ヒット案件」
- ルビ / 英字 / 記号の扱い: 未決定（ルビは濫用しない方針）
- 章題 / scene 名の形式: 未決定

## パロディ原則（canon）

- **狙う先**: ジャンルの文法（追放もの・なろう系・悪役令嬢・異能学園ものの共通テンプレ）
- **狙ってはいけない先**: 特定の実在作品・固有キャラクター・固有の設定語
- 判定基準: その表現を「どのジャンルの何テンプレか」で説明できるか。特定作品名でしか指せないならNG
- 例: 「女神役を三時間やらされた」は OK、「あの作品の女神役」は NG

## 独白の温度（canon）

- 主人公の独白は「嫌な理屈」のトーン。言い訳と本音が混ざる
- 合理化語彙（「つまり」「要するに」「思えば」等）が出たら drafter-preflight の self-check を通す
- 独白は身体反応と並走させる。観念だけで閉じない（剥奪された UI を今も探してしまう視線、など）

## 台詞の口語化ガイド（canon）

テーマをそのまま言っている台詞は臭い。ラストで効く台詞ほど、途中では雑で短く言いよどむ方がいい。

### 原則

- 主語を抜く
- 一文を短くする
- 途中で切る
- 同じことを二回言わない
- 感情は滲ませる（言い切らない）

### 実例バンク

| 説明的（NG 寄り） | 口語（OK 寄り） |
|---|---|
| 役は回せばいい。痛みも回せばいい | 同じやつにばっか割食わせんなって言ってんだよ / 役、回せ。泥もだ |
| 支えることと、消えることは違います | 支えるのはいい。勝手に消えるなって話 |
| 一人の固定犠牲を認めるわけにはいかない | ずっとそいつ一人に背負わせるのが終わってんだよ |
| チートや能力がなければ、ただの凡人だ | 剥いだら何も残んねえよ / だから今、素で立てって言ってんだ |

### 制度語のレジスター

本文の台詞で制度正式名称をそのまま使わない。砕いて言い換える。

| 正式名称 | 会話で使う | もっと砕ける |
|---|---|---|
| 能力核再付与権 | 復権 | 元に戻れる権利 / 返してもらえるやつ |
| 異端者 | （そのまま使える） | 痛ん者 |
| 役割破壊 | やらかした | 台本壊した |
| 成仏点 | 点 / 点数 | （そのまま使える） |

## 成長モデル（canon 方針）

- この作品の成長は **克服ではない**
- **症状の扱い方を少し覚える** 程度がちょうどいい
- パターン: また同じことをやる → でも前より少しだけマシ → 違う場面だとまた再発 → それでも前に進む
- きれいな克服は「更生教材」になってしまい、作品が弱くなる
- 「更生未満」が芯なので、成長はつねにノイズと再発を含む

## 会話温度のモード表（canon）

場面ごとに会話のモードを変える。ずっと乾いた皮肉だと読者が慣れる。

| 場面 | 温度 |
|---|---|
| 実習前 | だるい、雑、学生っぽい |
| 実習中 | 短い、仕事口調 |
| 帰還後 | 軽口多め、でも痛い |
| 深夜の寮 | 本音が少し出る |
| 教官面談 | 冷たい、逃げ場がない |
| 祭り回や変な案件 | かなり笑える |

## 3 層警告の文体バンク

実習中の警告は、以下の 3 層のいずれかで顕現する。drafter は層を混同しない。

### 層1: 個人通知（内なる声）

- 「注意。役割破壊 lv.1。速やかに配役へ復帰せよ」
- 視界の端に走るテロップ。読点なしの事務的な短文が基本

### 層2: 世界反応（物理現象）

- 膝から下が石化してくる
- 指先に火花が散る
- 声がエコーして自分の台詞に聞こえなくなる
- 地の文では「現象として起きた」ことを書く。原因説明は教官の台詞で後出し

### 層3: 帰還後評価（面談室）

- 学園に戻ってから、成仏点の減点と次勤務シフトの変更で告げられる
- 会話は短く、感情を殺した事務トーン

## 絶対 NG

- 役割語テンプレの貼り付け（"〜ですわ" を属性だけで与える等）
- 元主人公たちをただの愚か者として嘲笑うだけの書き方
- 和解・反省・改心で会話を閉じること
- 処罰を「奪っていたものの側」以外の因果で配ること
- 俺つえー批判を説教として書くこと
- 地の文での全知的なメタコメント（「こうして彼は知らないままだった」等）

## 避ける

- 過剰な情景描写（演出衝動と区別がつかなくなる）
- 説明セリフの連発
- 恋愛を"救い"として扱う描写（役を降りた後の関係として扱う）
- 「くそ」の乱発
- 勢いだけの決め台詞

## 口調サンプル

### 主人公（char-001）

- 本音に触れると「くそ」で切る
- 褒められると言い訳で否定する（「ちゃんとって言うな」「あれは……あいつが主人公だったからだ」）

### 相棒（char-002）

- 痛みを茶化してから「それは正常だ」と肯定する
- 「似合うじゃん。黙ってるし」のような軽口

### 世界観汚染常習者（char-003）

- 他作品風の決め台詞が反射的に出る
- 注意されると「違う違う、これは違うって！」と慌てる

### 主任教官（char-100）

- 言語化で追い詰める
- 「反省ではなく演出」を見抜く口癖
