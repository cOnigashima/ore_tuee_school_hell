## 1. この作品の会話の基本原則

### 基本方針

この作品の会話は、説明のためにしゃべらせない。しゃべる時は、必ずどれかをやる:

- 相手を刺す
- 自分を守る
- 空気をごまかす
- 責任を押しつける
- 本音が漏れる

### 理想温度

乾いた皮肉 6 : 生活のくだらなさ 2 : 不意の本音 2

ずっと重くしない。ずっと軽くもしない。笑えるのに、あとで少し効く温度。

### 会話の禁止傾向

- テーマを一息で全部言い切る
- 抽象語を重ねすぎる
- 誰もが同じレベルで物事を理解している
- 全員がきれいに言語化できる
- 説明が一往復で終わる

### 推奨傾向

- 途中で切る
- ごまかす
- 言い直す
- 冗談で逃げる
- 正しいけど感じ悪い言い方をする
- 本音ほど短い

## 2. 地の文の基本温度

### 視点の軸

基本は久瀬の近接三人称。ただし、久瀬は分析しすぎるので、地の文まで説明臭くならないようにする。

### 久瀬地の文のルール

- 長く考え込まない
- 比喩は少なめ
- 人を見る時にまず席や駒で見てしまう
- でもそれを自分で嫌がっている
- 本音はたいてい「くそ」に戻る

### 久瀬地の文の語彙

席 / 駒 / 壁 / 踏み台 / 養分 / 看板 / 前 / 裏 / 火消し / 穴埋め

### 久瀬地の文で避けたい語

役割 / 物語構造 / 消費社会 / システム / 共同体 / テーマ

→ 作者の頭の中では使っていい。でも久瀬の地の文では、もっと下品で具体的な言い方に落とす。

### 例

**悪い例**:
> 志摩は支援役として過剰に機能しており、構造的に搾取されやすい。

**良い例**:
> 志摩がまた裏を拾ってる。拾うやつが一人いると、場はきれいに回る。だから嫌なんだよ、それが。

## 3. 章ごとの会話温度

| 場面 | 温度 |
|------|------|
| **案件前** | だるい / 皮肉 / まだ他人事っぽい / 誰かが茶化す |
| **案件中** | 具体 / 短い / 命令形が増える / 感情は後回し |
| **帰還後** | 軽口が増える / でも痛みが混ざる / 本音は一個だけ漏れる |
| **凡日** | 一番くだらない / 一番素が出る / だから一番効く |
| **教官面談** | 逃げ場がない / 説明ではなく査定 / 事実を突きつける |
| **終盤** | きれいに喋らせない / 言い淀みを残す / 長文の決め台詞は最終局面だけ |

## 5. クラス12人の口調ガイド

### 1. 久瀬

- **一人称**: 俺
- **よく使う語**: 席 / 駒 / 壁 / くそ / 早い / 遅い / 面倒
- **あまり使わない語**: 希望 / 絆 / 大事 / 信じる
- **口調**: 短い。やや投げる。説明しそうになって自分で切る
- **感情が出る時**: むしろ語数が減る。本気で嫌な時ほど、単語で切る
- **代表台詞**: 「その席、切るの早すぎるだろ」「……また同じやつに食わせるのかよ」「くそ」

### 2. 柊透真

- **一人称**: 僕
- **よく使う語**: 一旦 / まだ / マシ / 確率 / そっち / 待て
- **あまり使わない語**: 熱い言い切り / 感情語 / 断定的な夢
- **口調**: 静か。論理的。でも本質的には怯えがにじむ
- **感情が出る時**: 逆に声が小さくなる。怒鳴らない
- **代表台詞**: 「一旦待て」「正解じゃない。そっちの方がマシってだけだ」「……それでも選ぶしかないのか」

### 3. 神代赫

- **一人称**: 俺
- **よく使う語**: 華 / 映える / 客 / 前 / 散る / 舞台
- **あまり使わない語**: 反省 / 協調 / 効率
- **口調**: 軽口。煽り。本音ほど冗談っぽく言う
- **感情が出る時**: 笑ってるのに笑えてない。語尾が少しだけ硬くなる
- **代表台詞**: 「映えるなら仕事だろ」「前に立つのは嫌いじゃない」「ずっと同じ傷で食うのは、なんか違う」

### 4. 志摩雫

- **一人称**: 私
- **よく使う語**: 回す / 引き受ける / 埋める / 継ぐ / 今 / ここ
- **あまり使わない語**: 熱血語 / 感傷 / 長い比喩
- **口調**: 端的。やや仕事口調。敬語寄りだが堅すぎない
- **感情が出る時**: 丁寧語が消えるより、文が短くなる
- **代表台詞**: 「それ、今は誰が拾いますか」「支えるのはいいです。消えるのは違います」「私なら回せます、が、一番危ないんですよね」

### 5. 朽葉理央

- **一人称**: 私
- **よく使う語**: 理屈 / その編成 / 事故率 / 正しい / 切れる / 無理
- **口調**: 丁寧だが冷たい。結論が先
- **感情が出る時**: 逆に少しだけ雑になる。そこがレア
- **代表台詞**: 「理屈としてはそっちです」「正しいことと、やっていいことは別でしょう」「分析はできます。配慮は、まだ訓練中です」

### 6. 比良坂陣

- **一人称**: 俺
- **よく使う語**: 人員 / 回す / 配置 / まず / そこ / 段取り
- **口調**: 指示語が多い。頼んでるつもりで命令になる
- **感情が出る時**: 自分が命令口調になってると気づいて、一瞬だけ詰まる
- **代表台詞**: 「人員回すぞ」「頼ってる。命令じゃない。……今のは命令っぽいな」「頭数で見てるつもりはない。見えてるってだけだ」

### 7. 百瀬由良

- **一人称**: 私
- **よく使う語**: はいはい / それ導線？ / うわ / やば / きつ / 嘘でしょ
- **口調**: 軽い。茶化す。でも本気になると急に真っ直ぐになる
- **感情が出る時**: 冗談が減る。むしろ真顔の一言が刺さる
- **代表台詞**: 「はいはい、それ導線？」「見えないからって、ないとは限らないの腹立つ」「きれいな言い方に変えただけじゃん、それ」

### 8. 冬城蓮

- **一人称**: 俺
- **よく使う語**: 減る / いくら / それ要る？ / 無駄 / 足りる / 配る
- **口調**: 細かい。数字が多い。損得感がすぐ出る
- **感情が出る時**: 逆に語が減って、「それ嫌だ」みたいな素の言い方になる
- **代表台詞**: 「それ、減るからな」「同じやつだけ削れてる方が嫌だ」「分けるのはいい。無駄にするな」

### 9. 七瀬聖

- **一人称**: 私
- **よく使う語**: 品位 / 祝福 / 継承 / 務め / 格式 / きれい
- **口調**: 丁寧。整っている。感情的でも言葉は崩れにくい
- **感情が出る時**: かえって語がきれいになりすぎる。そこが危うさ
- **代表台詞**: 「引くこと自体は悪ではないでしょう」「継ぐ席に意味があるのも事実です」「肩書がなくても拒否できる方が、たぶんましです」

### 10. 鷺宮鉄

- **一人称**: 俺
- **よく使う語**: もう一回 / それで伸びる？ / 地味 / 反復 / まだ / 行ける
- **口調**: 体育会系。勢いが先。細かい説明は嫌う
- **感情が出る時**: 勢いが止まり、妙に真面目になる
- **代表台詞**: 「それ、回数やるしかねえのかよ」「地味でも回るならやる」「伸び率じゃなくて持続で殴るってことか」

### 11. 弓月澪

- **一人称**: 私
- **よく使う語**: 嫌な感じ / 今日ズレてる / それ危なくない？ / 大丈夫？ / やばい / 無理かも
- **口調**: 不安が先に出る。少し縁起を気にする人の喋り
- **感情が出る時**: むしろ丁寧になる。怖い時ほど慎重に喋る
- **代表台詞**: 「それ、嫌な流れしてる」「大丈夫って言われるの一番困る」「保障なくても行くしかないの、ほんと嫌」

### 12. 藍沢灯

- **一人称**: 俺
- **よく使う語**: 本気なら / まあ / まだ / 俺の席じゃない / 真価 / たぶん
- **口調**: 斜に構える。逃げ道を残す。本気になるほど逆に静か
- **感情が出る時**: 言い訳が減る。そこが見どころ
- **代表台詞**: 「まあ、本気ならできるけど」「それ、俺の席じゃないんだよな」「ただの俺で行く。……それでいい」

## 6. 教官・先輩の口調ガイド

## 9. 会話を良くする実戦ルール

### ルール1: 一人だけきれいに喋らせる

一つの会話で、一人しかきれいに喋らせない。全員きれいだと説教になる。

### ルール2: 本音は短く

本音は短く。理屈は長くていい。刺さるのは短い方。

### ルール3: テーマは直接言い切らない

代わりに、駒 / 席 / 養分 / 継ぐ / 回す みたいな言葉でズラす。

### ルール4: ズレたまま終える

会話の最後は、和解じゃなくて少しズレたままで終えていい。この作品はその方が合う。

### ルール5: 軽さと痛みを混ぜる

ギャグ回でも、誰かの古傷が少しだけ触れている状態にする。逆に重い回でも、くだらない一言を消さない。
