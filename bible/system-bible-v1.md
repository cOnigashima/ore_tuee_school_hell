## 学園の正体

表向きは、異端者を再教育する学園。実態は、**配役矯正機関 兼 長期運用物語の保守工場**。

## 学園の場所

**幕間域**に存在する。どの物語世界にも属さない中間領域で、各実習世界へゲート接続する。

## 入学者の定義

学園へ送られるのは、以下に該当する者:

- 主役として暴走した者
- 他人の見せ場を食いすぎた者
- 固定席に適応しすぎた者
- 物語を壊した者
- 削除対象・再配置対象になった者

悪人だけではない。**うますぎて危険な者**も対象。

## 呼び名

- 正式: **異端者**
- 生徒の俗称: **痛ん者**

## 剥離

### 剥離式

入学時に**能力核**を封印する儀式。

### 封印対象

- チートスキル
- 加護 / 祝福
- 役職固定補正
- 鑑定UI
- 死に戻り / セーブ＆ロード
- 有能相棒リンク
- 有能部下指揮補正
- 並列思考 / 高速思考
- 無限アイテム / 召喚枠

### 原則

剥離後の学生は、基本的に「チートや能力がなければただの凡人」。

## 配役出力

学生は自前の能力を使えない。実習では案件ごとに**配役出力**が一時貸与される。

- その席を成立させるための最低限の性能
- 昔の最強能力そのものではない
- 個人の"俺つえー"ではなく、仕事用の出力

## 教官

### 原則

教官は**今も能力を使える**。しかも学生より明確に格上。

### 性質

教官能力は「校務権限」。無双よりも、監査・差し戻し・拘束・再演・配役固定に強い。

### 代表教官

| 教官 | 権限 |
|---|---|
| 平 | 査定・照合・差し戻し |
| 芹沢 | 再演・巻き戻し |
| 九条 | 配役固定・舞台拘束 |

### 制約

- 教官単独では能力核返還はできない
- 教官単独では永久固定席の確定もできない
- それは学園長級の権限

## 学園長級

以下のみ扱える:

- 剥離
- 返還
- 門管理
- 最終統合案件の割当
- 卒業解放の確定

## 実習中の痛み / 死 / 帰還

### 痛み

完全に本物。

### 怪我

- 軽傷: 帰還時におおむねリセット
- 重傷: 数日〜数週間の後遺痛が残る
- 精神負荷: 継続して残る

### 死

学生は実習中に死亡すると**幕間域へ強制帰還**。永久死ではない。

### 死亡時の代償

- 激しい断線痛
- 評価大幅減点
- 一時行動制限
- 繰り返すと**残響**が残る

### 残響

同じ痛みや同じ席に何度も馴染んだ結果残る後遺症:

- 幻痛
- 悪夢
- 特定席への過適応
- 感情の鈍化
- 退場癖、補助癖などの固定化

### 現地キャラ

現地世界の人物は原則として**本当に死ぬ**。実習世界は幻ではない。

## 実習世界の時間と記憶

### 時間

実習世界の時間は**継続して流れる**。再訪時、前回の結果は残っている。

### 記憶

現地レギュラーは、基本的に前回のことを覚えている。例外は、その世界自体がループ構造を持つ場合のみ。

## 学年

年齢ではなく**在籍年次**で区切る。

| 学年 | 名称 | テーマ |
|---|---|---|
| 一年 | 剥離基礎科 | 凡人として立つ年 |
| 二年 | 配役応用科 | 他人と噛み合わない年 |
| 三年 | 卒業実習科 | 復権と共同配役のどちらを選ぶかの年 |

## 在籍状態

| 区分 | 意味 |
|---|---|
| 在籍 | 通常学生 |
| 進級 | 次学年へ |
| 留年 | 単位不足 or 固着指数過多 |
| 仮継続席候補 | 固定席候補として査定中 |
| 継続在籍 | 卒業延期で固定席運用に入る状態 |
| 卒業解放 | 学園の配役管理から解放 |

## 留年理由

1. 主役食いが抜けない
2. 固定席適性が高すぎて外されない
3. 協働不能
4. 能力なしの日常適応に失敗

## 成仏

- **正式名称**: 卒業解放
- **生徒語**: 成仏
- 学園の配役管理から解放される
- 能力核は返還されない
- 元の"主役保護付き世界"には戻らない
- 記憶は保持
- ただの一人として生きる権利を得る
- 希望すれば、**同じ期の仲間と同じ生活圏へ移れる**

## 復権

### 意味

能力核再査定と再付与の総称。

### 三段階

| 段階 | 名称 | 内容 |
|---|---|---|
| 1 | 校務補助権 | 限定環境で一部使用可 |
| 2 | 準復権 | 一部能力の再解放 |
| 3 | 全復権 | 元の階層に近い再付与 |

### 学園のいやらしさ

学園はこれを全部まとめて「**戻れるかもしれない**」という餌として扱う。実際は、固定席に長く座った者ほど近づく。

## 評価指標

### 表向き（生徒 + 教官に見える）

| 柱 | 内容 |
|---|---|
| 配役点 | 席を守れたか |
| 成立点 | 物語を成立させたか |
| 協働点 | 使い潰さず連携できたか |
| 生活点 | 能力なしの日常を破綻させていないか |

### 裏指標（教官のみ）

| 指標 | 内容 |
|---|---|
| 固着指数 | 特定席に馴染みすぎていないか |
| 他者固定指数 | 他人を固定席へ押し込みやすいか |

## 案件の種類

- 個人案件
- ペア案件
- 班案件
- 公開案件
- 最終統合案件

## 週の流れ

| 曜日 | 内容 |
|---|---|
| 月 | 案件通達・授業 |
| 火 | 一次実習 |
| 水 | 査定・補講 |
| 木 | 再実習 / 班演習 |
| 金 | 評価確定・役替え |
| 土 | 自主訓練 / 観覧 / 校内行事 |
| 日 | **凡日**（何も起きない日を、ちゃんと生きる訓練日） |

## 生活

### 住居

- 一・二年: 原則二人部屋
- 三年: 準個室可

### 生活費

- 食事 / 寮 / 医療は学園持ち
- 小遣いは**生活点 + 案件手当**
- 減点・処罰で支給減

### 外出

- 凡日に一定条件で許可
- 処罰中は制限
- 他世界の通貨・アイテムは持ち帰れない

## 供養祭

学園の文化祭であり、**固定席なしで短い物語を成立させられるかを試す公開演習**。

- クラス単位の短編・連作が中心
- 他学年・教官・一部外部観覧あり
- 没席や削られた配役の"供養"の意味も持つ

## 本制度の核心

学園は、固定席を作ることで長く売れる話を安定運用する。しかもその見返りとして、復権という餌をぶら下げる。この制度に対して主人公たちは、**固定するのは人ではなく、看板と機能にする**という共同配役で対抗する。
