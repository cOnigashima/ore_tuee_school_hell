## purpose

読者にこの 3 つを渡す:

1. **この学園が何をさせる場所か** — 能力を剥がされた痛ん者たちが、他人の物語を成立させるために働かされる場所
2. **久瀬がどういう最低さと脆さを持つ主人公か** — 人をまず席や駒で見てしまう。しかも自分でもそれを嫌がっている
3. **この作品の温度** — 笑える。けど少し寒い。そして、その寒さがあとで効く

## ep01「ちゃんと負けろ」

### 話の仕事

世界観導入。久瀬主人公の確立。「この学園では、勝つことより、きちんと負けることが仕事になる」と示す。

### 目標

久瀬がノクス世界で「越えたくなる壁」として負ける。

### 障害

久瀬は、勝つ方が自然。しかも"綺麗に負ける"のではなく、"主役を立てるために負ける"のが嫌。

### 刺さる古傷

- 久瀬：主役食い
- 久瀬：自分が前に出ないと中身が空っぽになる恐怖

### 開示

- 学園は再配役施設
- 痛ん者という俗称
- 教官は強く、学生は能力なし
- 実習では配役出力のみ使える
- 久瀬は人を先に席で見てしまう

### 見せる絵

- 教室で処罰帰りの生徒がいる
- 平が案件を読み上げる
- ノクスの闘技場
- 久瀬の監査警告
- "ちゃんと負けたのに気持ち悪い"終わり方

### 変化

久瀬は初めて、「負けるのが嫌」ではなく "意味のある負け方をさせられるのが嫌" だと気づく。

### ラストの引き

減点 + 次回石像勤務。

### 3000 字配分

| 字数 | 内容 |
|------|------|
| 0–600 | 教室、痛ん者の空気、石像帰りの雑談 |
| 600–1200 | 平の案件説明、久瀬指名 |
| 1200–2200 | ノクス実習、壁として戦う |
| 2200–2700 | ちゃんと負ける |
| 2700–3000 | 帰還、石像処罰 |

### 会話温度

乾いた皮肉 7 : 説明 2 : 痛み 1

### NG

- 学園制度を一気に全部説明しない
- 久瀬を最初から共感しやすくしすぎない
- ノクス側キャラを濃くしすぎない

## ep02「石像勤務」

### 話の仕事

この作品の笑いと寒さを定着させる。処罰が比喩ではなく、本当に石像や背景になると見せる。

### 目標

石像勤務を終える。できれば何も考えずに終わらせたい。

### 障害

石像勤務は暇だから、考えてしまう。しかも「主人公が勝てますように」と祈られるたびに、自分が部材だったとわかる。

### 刺さる古傷

- 久瀬：見られないことの痛み
- 神代：目立たない席への嫌悪
- 志摩：支えるだけの存在にされる寒さ

### 開示

- 九条の配役固定権限
- 背景/石像/エフェクトが本当に処罰として存在する
- 生徒たちの軽口の空気
- アウレリアの"祈り席"の雰囲気を軽く触る

### 見せる絵

- 神殿入口の石像
- 祈る人々
- 石像なのに膝が痛い感覚
- 帰還後の教室の雑談

### 変化

久瀬は、"前に立たないだけ"じゃなく、他人の勝ちのために沈黙させられる席があると身体で知る。

### ラストの引き

次案件はグラム。しかも「追放する側」。

### 3000 字配分

| 字数 | 内容 |
|------|------|
| 0–800 | 九条による石像処罰 |
| 800–1800 | 神殿入口の石像勤務 |
| 1800–2400 | 祈りを聞く / 石像目線の寒さ |
| 2400–3000 | 帰還、教室、グラム案件告知 |

### 会話温度

くだらなさ 4 : 寒さ 4 : 哀愁 2

### NG

- 処罰をギャグだけで終わらせること
- 神代や志摩をこの話で前に出しすぎること
