# /draft

初稿を執筆する。

## 入力

`backlog/` にあるpitchファイル、または指定されたあらすじ

## 手順

1. 指定されたpitchまたは最新のpitchを読む
2. `bible/` の設定資料を参照する
3. `approved/` の直近3話を読み、文体と流れを把握する
4. 初稿を執筆する

## 執筆ルール

- `bible/rules.md` の文体・禁則に従う
- 一話あたり3000-5000字を目安
- 冒頭で前話との繋がりを意識する
- 末尾で次話への引きを作る

## 出力フォーマット

```markdown
# [タイトル]

[本文]

---

## メタ情報

- 文字数: XXX字
- 前話: [前話タイトル]
- 次話への引き: [要約]
```

## 保存先

`drafts/draft_YYYYMMDD_HHMMSS.md`
