# /pitch

次話のアイデアを複数提案する。

## 手順

1. `approved/` の公開済み原稿を読み、現在の進行状況を把握する
2. `arcs/` のプロットを確認し、今後の展開を把握する
3. `bible/` の設定資料を参照する
4. 次話の候補を6つ提案する

## 出力フォーマット

各候補には以下を含める:

```markdown
## 候補 N: [仮タイトル]

### あらすじ
[100-200字]

### 見どころ
[この話で読者に伝えたいこと]

### キャラクター
[主に登場するキャラクター]

### アーク上の位置づけ
[全体の物語におけるこの話の役割]
```

## 保存先

`backlog/pitch_YYYYMMDD_HHMMSS.md`
