## 作品の一文定義

かつて物語を食い潰す側だった"最強主人公"たちが、今度は物語を成立させるための裏方労働に回される学園。

## 作品設定文（企画書サイズ）

『元オレつえー学園』は、物語を壊した元・最強主人公たちが再配属される矯正学園である。生徒たちは各種の実習世界へ派遣され、かませ犬、ライバル、負けヒロイン、解説役、背景役、果ては神殿の石像や女神の声、必殺技の火花といった"物語を支える部材"として働かされる。役割破壊の処罰は重く、見せ場を奪った者は連続背景役、演出に酔った者は演出素材勤務へ落とされる。かつて世界の中心だった彼らは、他人の物語を成立させることを学ぶが、主役でいた頃の癖も誇りも簡単には消えない。笑って愚痴をこぼし、互いを雑に慰め合い、それでも本音に触れそうになると「くそ」と話を切る――そんな元最強たちの、更生未満の労働学園劇。

## 世界観の前提

- ただのファンタジー学園ではない。学園で訓練され、実習用の"物語空間"に派遣される構造
- 配役の粒度は、人物役 / 背景役 / 装置役 / 演出役まで及ぶ
- 少しだけ自然主義を捨てる。でないと「なぜ人間が火花をやっているのか」で読者が引っかかる

## 処罰体系（役割破壊に対する処罰）

**設計基準**: その処罰は、その人物が昔に奪っていたものの側に落ちているか。

### 背景勤務

見せ場を奪った者は、しばらく背景に落とされる。

- 神殿の石像
- 玉座の間の甲冑
- 学園の銅像
- 祝祭の群衆
- 王都の通行人
- 主人公の勝利を見守る無名兵

### 演出勤務

技を盛りすぎた者、決め台詞に酔った者は、技そのものの素材に回される。

- 必殺技の火花
- 斬撃の軌跡
- 爆煙
- 稲妻
- 覚醒時の光
- 勝利演出の風

### 神話勤務

神や運命を都合よく使った者は、今度は神話の部品になる。

- 女神の声
- 神託
- 聖剣の台座
- 封印の石碑
- 祝福の光
- 教会壁画の聖人

### 使い捨て勤務

一撃必殺・一話消費の快楽に寄りかかっていた者への処罰。

- 決め技の弾頭
- 囮
- 開幕一発で吹き飛ぶ中堅
- 主人公の格上げ要員
- 「こいつ強え！」と言うだけの先輩

### 因果の例

- 神の加護で無双してたやつ → 女神役
- 必殺技連発で格を潰してたやつ → 火花・爆煙
- 世界の中心ぶってたやつ → 背景石像
- 一撃で全部終わらせてたやつ → 弾頭

## 恋愛の扱い

- 普通の学園ラブコメにしない。強いのは、**役として与えられた恋愛ではなく、役から降りたあとに初めて始まる恋愛**
- 元ハーレム主人公は、一対一の関係が全然わかっていない
- 元負けヒロインは、「どうせ私は誰かを立てるための役だろ」と疑っている
- 元ライバル同士は、勝敗と無関係な好意をうまく扱えない
- 職員側は、恋愛が配役を乱すので警戒している

## 会話の温度（文体の核）

- 笑えるのに、少しだけ胸が痛い
- 苦労話は「反省」ではなく **禁断症状**
  - 主役を食いたい / でも食ったら罰せられる / 譲りたい / でも譲ると空っぽになる / 理屈はわかる / でも気持ちが追いつかない
- 和解で閉じない。少し本音が見えたところで、誰かが雑に切る
- 本音に触れると「くそ」で話を終わらす

### サンプル会話

```
「また背景役か」
「何だった」
「神殿の石像」
「似合うじゃん。黙ってるし」
「うるせえ」
「参拝客、何祈ってた」
「主人公の武運長久」
「……きついな」
「くそ、もう寝る」
```

```
「お前、今回ちゃんと負けたんだって？」
「ちゃんとって言うな」
「褒めてる」
「褒めるな。あれは……あいつが主人公だったからだ」
「うん」
「だから、別に俺は」
「うん」
「……くそ」
```

```
「女神役どうだった」
「三時間、知らない主人公に祝福を与えてた」
「泣いた？」
「笑った」
「そうか」
「笑ってから、ちょっとだけ死にたくなった」
「それは正常だ」
```

## 風刺の厚みを作る 3 点（痩せさせないために）

- なぜ人は俺つえーに快楽を感じるのか

→ ジャンルを見下す話ではなく、ジャンルの生産ラインに巻き込まれた連中の話にする。
